AI Snap – Smart Class Attendance System
Overview

AI Snap is an AI-powered smart attendance management system that automates classroom attendance using Face Recognition and Voice Recognition technologies. The system identifies students in real time through a webcam and also supports voice-based attendance verification for enhanced accessibility and security. AI Snap reduces manual work, prevents proxy attendance, and provides accurate attendance tracking for educational institutions.


Features
🎯 Real-time Face Attendance
🎤 Voice-based Attendance Verification
📊 Attendance Dashboard
🧠 AI & Machine Learning Integration
📁 Attendance Report Generation
🔒 Secure Student Authentication
⏱️ Fast and Automated Attendance Process
📈 Attendance Analytics & Monitoring
